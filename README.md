# AXT-sceneryImgSet

綾急技研メンバーが素材用に撮影したそのへんの景色の写真です。

## ファイル命名規則

『`10進数8桁のID`\_`撮影者名`\_`撮影場所`.`拡張子`』で揃えます。  
拡張子が揃っていない可能性がありますがそのへんは利用者がなんとかしてください。  
一般的な形式だと思うのでImagemagickとかで一括変換してあげればいいんじゃないですかね。

2022年12月16日現在、計 11 枚が保存されています。
